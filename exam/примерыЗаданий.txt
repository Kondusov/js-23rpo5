случайную ячейку таблицы. Повторное нажатие на кнопку должно закрашивать
новую незакрашенную ячейку, пока ячейки не закончатся.

При заходе на страницу сайта покажите юзеру обратный отсчет, который закончится
через час от момента захода юзера на сайт.
Сделайте так, чтобы после обновления или закрытия страницы,
наш счетчик продолжал работать.
Добавьте кнопку сброса счетчика.

Сложность 4:

Сверстайте HTML элемент, содержащий текст (минимум три слова).
Сделайте так, чтобы по клику на любое слово из этого текста,
это слово меняло порядок своих букв на обратный.

Напишите рандомайзер.Принимайте из HTML формы диапазон поиска,
вычисляйте рандомное число в этом диапазоне, и выводите на страницу.
Нужно исключить повторения!

Напишите функцию, которая принимает из HTML формы набор целых чисел,
(вводится в таком виде - 6,5,8,3,9). Далее функция работает с этим набором
и возвращает сумму всех чисел, кратных "3".
Результат выводим на HTML страницу.

Список дел (To-Do) с сохранением времени добавления.
Создай поле ввода и кнопку «Добавить».
При нажатии на кнопку текст задачи и Unix timestamp момента создания должны сохраняться в массив в localStorage.
При загрузке страницы список должен отрисовываться из хранилища.
Каждая задача должна отображать текст и время в формате: «Сделано в 14:30:05».

Дан абзац со словами и инпут. В инпут вводится слово. По клику на кнопку
найдите это слово в абзаце и покрасьте его фон в красный цвет.

Сложность 3:

Напишите функцию, которая принимает строку (получите из HTML элемента).
Так же функция принимает две буквы (из формы), какую заменить, и на что заменить.
И возвращает уже измененную строку.
Результат выводим на страницу.

Сделайте функцию, которая будет принимать число из html формы,
а возвращать это число прописью и вставлять на страницу.
Функция запускается по наступлению события (отправка формы).
Пусть функция работает с числами от 0 до 9.
Смотрите пример: func(5); // выведет 'пять'

Напишите функцию, которая узнает системное время.
Функция принимает из HTML формы - имя пользователя.
И в зависимости от текущего часа, приветствует пользователя сообщением
(доброе утро, добрый день, добрый вечер) + имя_пользователя, это сообщение вставляет на страницу.

Следить за курсором мыши, и выводить координаты на экран (постоянно обновлять).
Создать HTML элемент и писать на страницу о событии
(курсор пришел на элемент, или курсор покинул элемент).
